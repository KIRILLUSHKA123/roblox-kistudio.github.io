Interactions System

System of interactions with custom

GUI

element based on

ProximityPrompt

. It works on the principle of the standard ProximityPrompt, and only the GUI element changes. There is only one drawback of the system: phones cannot interact with this GUI element in any way. There is also an indicator that the button is pressed (in the second picture) - a line that stretches depending on how long you hold the button and is quickly removed if the pressing is stopped. It was done in about 20-30 minutes. This was done by

Kyrylo.
